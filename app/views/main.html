What is Days Off?

"Days Off" is a simple tool that allows supervisors to easily approve requests of days off made by employees. Even when they are not at their desk, managers can approve or deny requests using any desktop or mobile device.

How is it built?

This web application has been built with AngularJS. A similar mobile application is available as well. The server side application has been developed with Node.js and the Loopback framework.
